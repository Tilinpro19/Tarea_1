## Instrucciones para el IDE (Recomendado)
Este proyecto está configurado con **CMake**. Para ejecutarlo:
1. Abra la carpeta del proyecto en su IDE favorito (CLion o VS Code).
2. El IDE detectará automáticamente el archivo `CMakeLists.txt`.
3. Haga clic en el botón de **"Run"** (triángulo verde).
